## BLAS and Windows

- NumPy and everything that depends on it (i.e., SciPy, pandas, scikit.learn)
is hard to compile on Windows because it uses libblas.
- NumPy has been switched over to OpenBLAS. Hopefully other libraries will
follow soon...
